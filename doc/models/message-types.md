# Message Types

Message types in LangChain are role-based objects used to structure inputs and outputs for chat models

<br/>

`HumanMessage`: Represents input from the end-user. These messages contain the questions, prompts, or interactions directed at the model

`AIMessage`: Represents the response generated by the AI model. It contains the text output, metadata like token usage, and requests to execute specific tools

`SystemMessage`: Sets the behavior, context, and tone for the AI. It is usually passed at the very beginning of a conversation to prime the model (e.g., "You are a helpful assistant")

`ToolMessage`: Stores the output or results after your code has executed an external tool (like a weather API or a search engine). This message is passed back to the model so it can formulate a final answer based on the tool's output
